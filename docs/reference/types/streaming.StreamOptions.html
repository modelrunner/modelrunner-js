The stream API options. It requires the API input and also offers configuration options.

Type Parameters

  • Input

Type declaration

  • Optional Readonlyaccept?: string

    The content type the client accepts as response. By default this is set to text/event-stream.

  • Optional ReadonlyautoUpload?: boolean

    Whether it should auto-upload File-like types to modelrunner's storage or not.

  • Optional ReadonlyconnectionMode?: StreamingConnectionMode

    The streaming connection mode. This is used to determine whether the streaming will be done from the browser itself (client) or through your own server, either when running on NodeJS or when using a proxy that supports streaming.

    It defaults to server. Set to client if your server proxy doesn't support streaming.

  • Optional ReadonlyqueryParams?: Record<string, string>

    The query parameters to be sent with the request.

  • Optional Readonlysignal?: AbortSignal

    The signal to abort the request.

  • Optional Readonlytimeout?: number

    The maximum time interval in milliseconds between stream chunks. Defaults to 15s.

  • Optional Readonlyurl?: string

    The endpoint URL. If not provided, it will be generated from the endpointId and the queryParams.
