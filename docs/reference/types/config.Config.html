Type declaration

  • Optionalcredentials?: string | CredentialsResolver

    The credentials to use for the modelrunner client. When using the client in the browser, it's recommended to use a proxy server to avoid exposing the credentials in the client's environment.

    By default it tries to use the MODEL_RUNNER_KEY environment variable, when process.env is defined.

    See

    • https://modelrunner.ai/docs/model-endpoints/server-side
    • #suppressLocalCredentialsWarning
  • Optionalfetch?: FetchType

    The fetch implementation to use for the client requests. By default it uses the global fetch function.

  • OptionalproxyUrl?: string

    The URL of the proxy server to use for the client requests. The proxy server should forward the requests to the modelrunner api.

  • OptionalrequestMiddleware?: RequestMiddleware

    The request middleware to use for the client requests. By default it doesn't apply any middleware.

  • OptionalresponseHandler?: ResponseHandler<any>

    The response handler to use for the client requests. By default it uses a built-in response handler that returns the JSON response.

  • OptionalsuppressLocalCredentialsWarning?: boolean

    Suppresses the warning when the modelrunner credentials are exposed in the browser's environment. Make sure you understand the security implications before enabling this option.
